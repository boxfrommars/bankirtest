тестовое задание для bankir.ru
цель: создать как можно более бессмысленное приложение с использованием:
    zend framework
    postgresql
    zend framework lucene
и небольшим налётом ajax


приложение "99 бутылок" представляет собой 3 коллекции: бутылок, напитиков, и напитков разлитых по бутылкам
пользователь может
    - добавлять, редактировать, а также удалять напитки, разливать напитки по бутылкам,
    - производить поиск по напиткам
    - "петь песню" про бутылки на стене (в это время бутылки падают и НЕ удаляются из коллекции)
