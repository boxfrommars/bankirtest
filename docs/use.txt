99 бутылок. Инструкция по эксплуатации
======================================


0. Описание
===========

Приложение "99 бутылок" представляет собой 3 коллекции: бутылок, напитиков, и напитков разлитых по бутылкам
пользователь может
    - добавлять, редактировать, а также удалять напитки, разливать напитки по бутылкам,
    - производить поиск по напиткам
    - "петь песню" про бутылки на стене (в это время бутылки падают и НЕ удаляются из коллекции)


1. Состав приложения
====================

Приложение 99 бутылок представляет собой сайт из 4-х разделов:
    
    1.1 Главная страница
    
        На главной странице находится область стола с расставленными на нём
        заполненными бутылками. Пользователь может "спеть песню" про бутылки,
        нажав на кнопку "sing!" под областью. чтобы остановить песню
        нужно нажать "stop!". Для повторения песни "once more"
     
    2.1 Страница напитков
        
        Здесь можно посмотреть список доступных напитков, напитки по отдельности,
        добавить новый напиток или отредактировать имеющийся (со страницы этого напитка).
        Также можно удалить любой напиток.
        
    3.1 Страница бутылок
        
        Здесь можно посмотреть список доступных бутылок. а также посмотреть
        отдельные бутылки с их описанием. Редактировать, добавлять бутылки нельзя
       
    4.1 Страница наполненных бутылок
    
        Здесь можно посмотреть список наполненных бутылок (наполненная бутылка
        состоит из бутылки, напитка и этикетки). Также можно посмотреть какая бутылка
        или напиток входит в состав наполненной бутылки (информация появится без
        перезагрузки - используется ajax)
        
        Также здесь можно добавить или удалить наполненную бутылку
        
На каждой странице доступна форма для поиска по напиткам.
